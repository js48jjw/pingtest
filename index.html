네트워크 안정성 테스트 정보

이 웹사이트는 Google.com 서버와의 핑(Ping) 시간을 측정하여 네트워크의 안정성을 확인합니다.

측정 항목 설명

  • 현재 핑: 가장 최근에 측정된 Google.com 서버와의 응답 시간
  • 평균 핑: 지금까지 측정된 핑 시간의 평균값
  • 최대 핑: 지금까지 측정된 핑 시간 중 가장 높은 값
  • 그래프: 시간에 따른 핑 시간 변화를 시각적으로 표시

네트워크 안정성 판단 기준

  • 안정적: 평균 핑이 50ms 이하이고, 큰 변동이 없는 경우
  • 양호: 평균 핑이 50-100ms 사이이고, 작은 변동이 있는 경우
  • 불안정: 평균 핑이 100ms 이상이거나, 큰 변동이 있는 경우

테스트 방법

  • 테스트 시작 버튼을 누르면 Google.com 서버에 1초 간격으로 핑을 보냅니다.
  • 20회 핑 체크 후 자동으로 테스트가 종료됩니다.
  • 실시간으로 핑 시간과 그래프가 업데이트됩니다.
  • 상태 표시등을 통해 현재 네트워크 상태를 직관적으로 확인할 수 있습니다.

기술 정보

  • Google.com 서버와의 ICMP Echo Request를 통해 핑 시간을 측정합니다.
  • 배경 이미지는 Picsum Photos에서 제공되며, 매일 오전 6시에 변경됩니다.
